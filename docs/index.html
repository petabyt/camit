A better way to control digital cameras

Camit is an app that can connect to your digital camera.

Features:

  • Support for Canon EOS cameras
  • Liveview with zoom
  • Profiles - preconfigured settings & parameters for each camera / workflow
  • Rich scripting API - add buttons and scripts to the app to do whatever you want
  • Get & Set ISO, aperture, shutter speed, white balance
  • Take pictures and BULB shots
  • Preview and download captured images
  • Manually and precisely focus lens
  • File manager and photo gallery
  • Magic Lantern integration
  • Capture and preview photos

Planned Features:

  • Support for Sony cameras
  • Use Android device as external microphone
  • GPS & clock sync
  • Customizable Zebras/histogram overlays

Camera as a Service (CaaS)

Remotely control a camera from anywhere in the world with a rich API over the web. Available for astronomy and long-term timelapse.
